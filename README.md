# Tagger-Panel Only

Panel-only edition of the Holorun Tagger Chrome extension.

## What this edition does

- Keeps the floating FAB and panel UI.
- Preserves tagging, restore, and panel workflows.
- Disables fullscreen overlay mode.

## Install (Unpacked)

1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this `Tagger-Panel-Only` folder.

## Files

- `manifest.json` - MV3 extension config.
- `background.js` - context menu and background wiring.
- `inject.js` - content script + panel logic.
- `smart-ui.css` - extension UI styles.
- `icon-img/` - extension images.
